# Sistema Hotel

Este projeto foi desenvolvido como uma atividade proposta pelo professor da disciplina de Programação Web. O objetivo da atividade é simular a criação de um site simples para um hotel, aplicando na prática os conceitos de estruturação e estilização aprendidos em sala de aula.
